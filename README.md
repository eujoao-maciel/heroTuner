<p align="center">A modern web application for tuning electric and acoustic guitars with real-time pitch detection and visual feedback. <a href="https://hero-tuner-client.vercel.app/" target="_blank">Live Demo</a></p>


<h2 align="center" id="About">About</h2>
<p align="start">
HeroTuner is a browser-based guitar tuner built to provide fast and accurate tuning. It uses the device microphone to detect the played note in real time, guiding the user with a responsive needle, tuning progress indicator, and visual feedback until the string is perfectly tuned.
</p>

<h2 align="center" id="Features">Features</h2>
<table align="center">
   <tr>
      <td>Real-time pitch detection</td>
      <td>Automatic & Manual modes</td>
      <td>Needle indicator</td>
      <td>Tuning progress</td>
      <td>Audio feedback</td>
   </tr>

   <tr>
      <td>Detects guitar notes through the microphone with low latency</td>
      <td>Automatically identifies strings or allows manual selection</td>
      <td>Displays tuning direction and accuracy with a smooth animated needle</td>
      <td>Shows progress while the string remains in tune</td>
      <td>Plays a confirmation sound when the string is successfully tuned</td>
   </tr>
</table>
